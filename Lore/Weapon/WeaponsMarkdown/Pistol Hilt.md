# Pistol Hilt

While the Jedi of old disdained blasters, modern Force users tend to need every advantage they can get.  More information about the attachment available on the Wookieepedia article.  PISTOL HILT This attachment can only be applied to lightsabers.[3]The user must spend a maneuver to switch from one weapon mode to the other (from lightsaber function to pistol function, for example). Models Include: None; users must custom-make each lightsaber attachment. Base Modifiers: When the lightsaber is not ignited, the wielder may fire the pistol using the following profile: (Ranged [Light]; Damage 6; Critical 3; Range [Short]; Stun Damage). Modification Options: None.
