# Model 134a Buzzsaw Rotary Cannon

Loud, heavy, and dangerous, the Model 134a is among the heaviest portable slugthrowers. More information about the weapon available on the Wookieepedia article.MODEL 134a BUZZSAW ROTARY CANNON[1][2][]The Model 134a can only be fired using the Auto-fire quality. Game Masters may spend three threat results or a despair result to have the weapon overheat and jam. When a Buzzsaw jams, the user must perform a maneuver to clear the jam before it can be fired again.[1][3]
