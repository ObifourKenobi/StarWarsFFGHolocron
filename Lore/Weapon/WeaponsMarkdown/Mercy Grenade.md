# Mercy Grenade

The Mercy grenade is a unique gas-based stun grenade which is typically used by large corporations and by covert agencies via shell corporations. More information about the weapon available on the Wookieepedia article.HIC MERCY GRENADE[1][2][]A character hit by a Mercy grenade must make a Hard difficulty Resilience check to avoid having an incomplete or inaccurate memory of events that transpire during that encounter. Additionally, they become disoriented for one round per threat result on the check. TRAPS AND TRIPWIRES[2][]As with frag grenades, a character can set [...] Mercy grenades [...] to detonate on impact, set them with a timer to detonate up to three rounds after being activated, or set up the explosive with connected a pressure detonator.
