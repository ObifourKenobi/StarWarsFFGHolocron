# Engineer's Hammer

An engineer's hammer is an all-purpose construction and demolitions tool.   More information about the gear available on the Wookieepedia article. ENGINEER'S HAMMER Using an engineer's hammer adds two boost dice to any checks made to build or demolish structures. When used as a weapon, an engineer's hammer has the profile as mentioned above.
