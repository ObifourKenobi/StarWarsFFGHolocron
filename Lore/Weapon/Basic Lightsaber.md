# Basic Lightsaber

The basic lightsaber is a simple but exceedingly elegant weapon.  Models Include: None; each lightsaber is individually crafted.  CLONE WARS ERA This lightsaber is a basic lightsaber hilt containing an unmodded Ilum lightsaber crystal. This crystal occupies two of the weapon's hard points. Lightsabers cannot be Sundered.
