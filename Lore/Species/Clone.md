# Clone
## Special Abilities
Clones start the game with one rank in Knowledge (Warfare) and one rank in Resilience. They still may not train these skills above rank 2 during character creation.
## Additional information
- **Kamino Training**: Clones start with one rank in the Physical Training talent.
