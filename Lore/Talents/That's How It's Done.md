Upon a successful skill check, the character may suffer 1 strain. If so, a number of allies within short range equal to the character's [[Willpower]] add an automatic #StarWarsAdvantage when making the same skill check until the start of the character's next turn. The range at which That's How It's Done can affect allies increases by one band per rank of That's How It's Done beyond the first.
