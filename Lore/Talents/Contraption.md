Once per game session, the character can take a Contraption action. With a successful #StarWarsDifficulty #StarWarsDifficulty #StarWarsDifficulty  Mechanics check, the character can fashion a device to solve a current problem using the tools and parts on hand. This device must be approved by the GM and functions for the duration of the current encounter only. Those tools are destroyed in the process. A #StarWarsTriumph result may be spent to allow the tools to be recovered to be used later.
