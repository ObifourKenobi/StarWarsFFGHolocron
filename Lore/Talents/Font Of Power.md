Force talent. Once per game session, the character may perform the Font of Power action. For the remainder of the encounter, if the character is a light side user, whenever another Force user within medium range rolls a #StarWarsForce, add an automatic light point to the results. If the character is a dark side Force user, whenever another Force user within medium range rolls a #StarWarsForce, add an automatic dark point to the results.
