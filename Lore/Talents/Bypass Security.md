The character removes a #StarWarsSetback per rank of Bypass Security from **their** Skulduggery and Computers skill checks made to disable a security device or open a locked door.

The following is a homebrew expansion of this talent from a community-produced homebrew sourcebook!

Improved
Activation: Passive
Ranked: No
Trees: Slicer (reSpec)


The character may spend three #StarWarsAdvantage or a #StarWarsTriumph result from checks to unlock doors or disable a security device to add #StarWarsBoost equal to ranks of Bypass Security to similar checks in the same system or location this session. The Game Master determines how similar a check must be to qualify.
