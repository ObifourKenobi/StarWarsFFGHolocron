Through their strong connection to the Force, some Force-sensitives can stretch their normal visual senses far beyond what would otherwise be possible.

# Basic Power
In its most basic form, Farsight allows the Force user to see in situations in which normal visual detection would be impossible, such as in total darkness or heavily obscuring smoke. Using Farsight, it is even possible for those without eyes or have lost their sight to see with perfect vision.

The user may spend a #StarWarsLightDark  to see out to medium range for the remainder of the round (or one minute in narrative time), and can see everything most sentients can see on a well-lit day. This functions even if the user normally can't see due to darkness or blindness.

# Upgrades
**Control Upgrade:** When making a Vigilance or Perception check, the user may roll a Farsight power check as part of the pool. The user may spend a #StarWarsLightDark  to gain a success or advantage result (character's choice) on the check.

**Control Upgrade:** The user may spend a #StarWarsLightDark  to see through one object at medium range as though it were transparent. The user may activate this multiple times, increasing the number of objects by one each time.

**Control Upgrade:** Spend a #StarWarsLightDark  to make out fine details on one object within medium range. The user may activate this multiple times, increasing the number of objects by one each time.

**Control Upgrade:** Spend a #StarWarsLightDark  to see microscopic details of one object within engaged range. The user may activate this multiple times, increasing the number of objects by one ach time.

**Control Upgrade:** This power gains the ongoing effect: Commit a Force die. The Force user's rank in the Perception skill by 1 (to a maximum of 5) each time this is performed. 

**Control Upgrade:** The user may spend two #StarWarsLightDark  to see in every direction simultaneously, noticing and observing things in a full 360-degree arc. The user may not activate this multiple times. 

**Duration Upgrade:** The user may spend a #StarWarsLightDark  to increase the duration by a number of rounds (or minutes in narrative time) equal to Duration upgrades purchased.

**Mastery Upgrade:** The user may spend two #StarWarsLightDark . The user now can see as though from a spot within close range (planetary scale) of the user's body. If the user is on the surface of a world, this distance can be a vertical altitude in the air as well as along the surface; if in space this can be in any direction. This effectively moves the character's personal point of view far away from the location of the user's body, allowing a viewpoint as if controlling a remote viewing mechanism.

**Range Upgrade:** The user may spend a #StarWarsLightDark  to increase the power's range by one range band equal to Range upgrades purchased. The user may activate this multiple times, increasing the range by this number each time.
